# Ledger

A proof-of-concept double-entry ledger system for iGaming platforms, built with Elixir and TigerBeetle.

This repository accompanies the Medium article: **"Building a Double-Entry Ledger with Elixir and TigerBeetle"**

## Quick Start

```bash
# Start TigerBeetle
./start_tigerbeetle.sh

# Run tests
mix test
```
