# Ledger

A proof-of-concept double-entry ledger system for iGaming platforms, built with Elixir and TigerBeetle.

This repository accompanies the Medium article: **[Building a Double-Entry Ledger with Elixir and TigerBeetle](https://medium.com/@altuntasfatih42/building-a-double-entry-ledger-with-elixir-and-tigerbeetle-f0f9fcc37408)**

## Quick Start

```bash
# Start TigerBeetle
./start_tigerbeetle.sh

# Run tests
mix test
```
